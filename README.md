# happiness_analysis
## Overview
Exploratory data analysis of global happiness factors and their impact on happiness scores.

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn, Statsmodels

## What I did
- Cleaned dataset and handled missing values
- Performed statistical analysis
- Visualized happiness by region
- Analyzed correlation between features
- Created multiple plots and insights

## Result
GDP, health, and social support strongly affect happiness

## Conclusion
Happiness is influenced by economic and social factors.
